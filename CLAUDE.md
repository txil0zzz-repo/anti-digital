# CLAUDE.md

This file provides guidance to Claude Code when working in this repository.

## Project Overview

Digital product store on Etsy. Selling printable/digital downloads — planners, budgeting sheets, ADHD notebooks, and living plans.

- **Etsy store**: TBD (new store, separate from ByTheFern)
- **Product type**: Digital downloads (PDF, Canva templates)
- **Niche**: Productivity, ADHD, budgeting, wellness

## Folder Structure

```
anti DIGITAL/
├── designs/
│   ├── planners/     — daily/weekly/monthly planners
│   ├── budgeting/    — budget trackers, expense sheets
│   └── adhd/         — ADHD planners, focus tools, habit trackers
├── listings/         — Etsy titles, descriptions, tags per product
├── templates/        — exported PDFs ready for upload
└── CLAUDE.md
```

## Workflow

1. Design in Canva → export as PDF
2. Save PDF in `templates/`
3. Write listing copy in `listings/` (title, description, tags)
4. Upload manually to Etsy as digital download

## Notes

- No Printify needed — Etsy handles digital delivery automatically
- Price range: €3–€15 per product
- Always include both A4 and US Letter sizes when possible
